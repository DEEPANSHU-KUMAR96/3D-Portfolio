# 🚀 Deepanshu Portfolio

A modern and interactive 3D developer portfolio built with React, Three.js, React Three Fiber, Framer Motion, and EmailJS.

## 🌐 Live Demo

🔗 Live Website: https://3-d-portfolio-bice-nine.vercel.app/

## 📌 Overview

This portfolio showcases my skills, projects, and experience as a MERN Stack Developer through an immersive 3D web experience.
It combines modern UI design, smooth animations, and interactive 3D models to create an engaging user experience.

## ✨ Features

* Interactive 3D Developer Workspace
* Smooth Animations with Framer Motion
* Responsive Design
* Project Showcase Section
* Skills & Technologies Section
* Contact Form with EmailJS Integration
* Modern UI/UX
* Optimized Performance
